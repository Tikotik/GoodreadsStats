# Bookworm Statistics

> Discover bookworm in you.

Statistics based on Goodreads.com profile.

## Discover read books:
 - Books count
 - Number of pages
 - Average book length
 - Average speed
 - Fastest book
 - Slowest book
 - TOP 10 - longest, fastest, authors, periods, languages...
 - Charts
 - etc.
